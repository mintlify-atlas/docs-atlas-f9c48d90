## What is Yggdrasil?

Yggdrasil is the monorepo that powers the student association **IFI-Navet** at the Institute of Informatics, University of Oslo. It provides a best-in-class experience for students attending company events through [ifinavet.no](https://ifinavet.no).

The project leverages modern web technologies including Next.js, TypeScript, Convex, and Clerk to deliver real-time, scalable services for event management, student engagement, and administrative operations.

## Technology Stack

Yggdrasil is built using a modern, type-safe stack:

- **[Next.js](https://nextjs.org/)** - React framework with App Router
- **[TypeScript](https://www.typescriptlang.org/)** - Type safety across the monorepo
- **[Convex](https://www.convex.dev/)** - Real-time database with sync engine
- **[Clerk](https://clerk.com/)** - Authentication and authorization
- **[Turborepo](https://turbo.build/)** - Monorepo build system
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first styling
- **[shadcn/ui](https://ui.shadcn.com/)** - Accessible React components
