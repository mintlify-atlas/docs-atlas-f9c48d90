## Overview

The database schema is defined in `packages/backend/convex/schema.ts` using Convex's schema definition API.

## Schema Definition

```typescript
import { defineSchema, defineTable } from "convex/server";
import { v } from "convex/values";
```

## Access Roles

The schema defines a union type for access control:

```typescript
export const accessRoles = v.union(
  v.literal("super-admin"),
  v.literal("admin"),
  v.literal("editor"),
  v.literal("internal"),
);
```

## Using Indexes

Indexes enable efficient queries. Example usage:

```typescript
// Query using an index
const student = await ctx.db
  .query("students")
  .withIndex("by_userId", (q) => q.eq("userId", userId))
  .first();

// Search using a search index
const companies = await ctx.db
  .query("companies")
  .withSearchIndex("search_name", (q) => q.search("name", searchQuery))
  .take(10);
```
