## Introduction

Bifrost is the official admin dashboard and backend for [ifinavet.no](https://ifinavet.no). It serves as the central hub for managing content, events, job listings, and other administrative tasks for the student organization IFIs Navet.

The application is built with Next.js and provides a comprehensive admin interface for managing all aspects of the student organization's digital presence.

### Core Technologies

- **Framework**: Next.js 16.1.6 with React 19
- **Language**: TypeScript 5.9
- **Backend**: Convex (real-time database and backend)
- **Authentication**: Clerk
- **Styling**: Tailwind CSS 4.1 with Shadcn/ui components
- **Rich Text Editor**: Tiptap (headless ProseMirror wrapper)
- **Validation**: Zod for schema validation
- **State Management**: Zustand
- **Analytics**: PostHog
- **Error Tracking**: Sentry

### UI Components

- **Component Library**: Shadcn/ui built on Radix UI primitives
- **Icons**: Lucide React
- **Forms**: TanStack Form with React Hook Form integration
- **Tables**: TanStack Table
- **Charts**: Recharts
- **Notifications**: Sonner for toast messages

<Note>
  Bifrost uses the App Router with Next.js 16 and React 19, leveraging the latest features including Server Components and Server Actions.
</Note>

## Development

### Prerequisites

- Node.js v20 or later
- pnpm package manager
- Clerk account for authentication setup

### Local Setup

```bash
# Navigate to the bifrost app
cd apps/bifrost

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your Clerk credentials

# Run the development server
pnpm dev
```

The application will be available at [http://localhost:3001](http://localhost:3001).

<Note>
  The application requires a Clerk account for authentication. Set up your Clerk application and add the credentials to `.env.local`.
</Note>
